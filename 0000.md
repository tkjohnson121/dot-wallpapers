Wallpapers for my [dotfiles](https://github.com/tkjohnson121/dotfiles.git);

`git clone git@github.com:tkjohnson121/dot-wallpapers.git ./.local/share/wallpapers`

Run this command in the current folder to number the images
`num=0; for i in *; do mv "$i" "$(printf '%04d' $num).${i#*.}"; ((num++)); done`
See this SO thread for more: https://stackoverflow.com/a/37826687
